00120 /*
00121         Like the syscalls, this returns the number of bytes on success.
00122         Unlike the syscalls, it returns 0 rather than an error
00123         when no bytes are available after interrupt (EINTR)
00124         or on non-blocking IO (EAGAIN); i.e. you may try later.
00125         On error, a negative value is returned.
00126         On errors that render the file unusable,
00127         it is closed for input or output, resp.
00128         These are most errors but EFAULT (bad buf)
00129         and ESPIPE (which clears the SEEK bit).
00130         For files that do not have the SEEK bit set,
00131         EOF is considered such an error and -LERR_EOF is returned.
00132         If you want to stream a disk file like 'tail -f', set the SEEK bit.
00133 */
